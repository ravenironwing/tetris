Novris by Raven Ironwing
Mutant Python Games (TM)

A simple tetris like game based on all continuous block sets that can be made in a 3x3 grid up to 5 cells in conplexity.

I used Tetris by "Kevin Chabowski"<kevin@kch42.de> as a starting point for this game.
